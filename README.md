# Novo Industries Assessment Tools

Professional accessibility and home safety assessment tools for Canadian municipalities, healthcare partners, and families.

Built and maintained by Novo Industries Inc., an Alberta corporation based in Calgary.

## The Tools

### RHFAC Assessment Tool (novo_rhfac_app_v1.1.html)

A browser-based facility assessment tool aligned with the Rick Hansen Foundation Accessibility Certification (RHFAC) v4.0 methodology. Twelve assessment categories. Report-type selector for Municipal, Private Sector, Healthcare, Educational, and General engagements. Municipal reports include a Next Dollar Impact section with a priority-ranked remediation table and a two-year capital pathway.

### Senior Home Safety Assessment Tool (shs_assess_v1.html)

A browser-based dwelling assessment tool aligned with CSA B652:23 Accessible Dwellings. Ten assessment categories. Dual output: a clinical summary for Occupational Therapists and a plain-language client report for families. Home Safety Score from zero to one hundred, findings ranked by fall-risk severity, and integrated navigation of Alberta and federal home modification funding programs.

## How to Use

Both tools are single-file HTML applications. Open the file in any modern browser (Chrome, Safari, Firefox, Edge). No server, no install, no account required. Data is stored locally in the browser. Export a JSON backup before clearing the browser or switching devices.
